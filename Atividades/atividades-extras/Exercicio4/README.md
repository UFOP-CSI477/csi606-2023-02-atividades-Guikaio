# Quarto exercício

- Este exercício consiste em introduzir a consulta em APIs de terceiros para utilizar os dados em suas aplicações. Você pode fazer alterações de estilização utilizando o arquivo `css/index.css`. Você deve continuar a implementação da funções `showError` e `insertRow` que estão no arquivo `js/index.js`. Também leia a função `getRepository`, nela é feita a busca na API do github utilizando o `fetch`. Siga as instruções contidas no arquivo `js/index.js` e sinta-se a vontade para tirar dúvidas na monitoria !
